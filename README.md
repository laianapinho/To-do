TODO App – Kotlin + Jetpack Compose

Aplicativo Android desenvolvido em Kotlin utilizando Jetpack Compose para construção da interface declarativa e arquitetura MVVM para separação de responsabilidades.

O projeto implementa gerenciamento de tarefas com:

✔ Criação e exclusão de tarefas

✔ Marcação de tarefas como concluídas

✔ Filtros (todas, pendentes e concluídas)

✔ Controle de estado da interface utilizando UiState (Loading, Empty, Success, Error)

✔ Organização em camadas (UI, Model e ViewModel)

A aplicação demonstra boas práticas de arquitetura, gerenciamento de estado e construção de interfaces reativas modernas no Android.
